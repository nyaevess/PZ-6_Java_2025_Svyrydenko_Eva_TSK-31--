№1. Що таке immutable об’єкт? Для чого об’єкти класу String зробили імутабельними? Чому клас String задекларований як final?

Immutable (незмінний) об’єкт — це об’єкт, стан якого не можна змінити після створення.
Будь-яка “зміна” створює новий об’єкт, а не змінює старий.

Об’єкти класу String зробили незмінними, тому що:

це підвищує безпеку (наприклад, рядки часто використовуються для паролів, URL тощо);

покращує продуктивність через можливість кешування (String Pool);

забезпечує потокобезпечність (не потрібно синхронізації при спільному доступі).

Клас String задекларований як final, щоб його не можна було наслідувати і змінити поведінку, що могло б порушити вищенаведені властивості.

№2. Що таке регулярні вирази? Наведіть приклади регулярних виразів.

Регулярний вираз (Regular Expression, regex) — це шаблон, який використовується для пошуку або перевірки тексту на відповідність певному формату.

Приклади:

[A-Za-z]+ — одна або більше літер (слово);

\d+ — одна або більше цифр;

[A-Za-z0-9]+ — комбінація літер і цифр;

^[A-Z][a-z]+$ — слово, яке починається з великої літери;

\s+ — один або більше пробілів.

№3. В чому полягає різниця між оператором «==» та методом «equals()»?

== перевіряє, чи два посилання вказують на той самий об’єкт у пам’яті.

equals() перевіряє, чи значення об’єктів однакові (залежить від реалізації методу).

Приклад:3. В чому полягає різниця між оператором «==» та методом «equals()»?

== перевіряє, чи два посилання вказують на той самий об’єкт у пам’яті.

equals() перевіряє, чи значення об’єктів однакові (залежить від реалізації методу).

Приклад:

String a = new String("hello");
String b = new String("hello");

System.out.println(a == b);      // false (різні об’єкти)
System.out.println(a.equals(b)); // true  (однаковий зміст)

№4. Для чого потрібні класи StringBuilder та StringBuffer?

Вони потрібні для зміни рядків без створення нових об’єктів (на відміну від String).

StringBuilder — швидкий, але не потокобезпечний (для однопоточних програм).

StringBuffer — повільніший, але потокобезпечний (використовується у багатопоточних програмах).

Приклад:

StringBuilder sb = new StringBuilder("Hello");
sb.append(" world");
System.out.println(sb); // Hello world****
